# Dotfiles

A carefully maintained collection of personal configuration files designed to provide a consistent, efficient, and reproducible development environment across systems.

## Contents

- Shell (`zsh` / `bash`)
- Git
- Neovim
- tmux
- Terminal
- Scripts

## Installation

```bash
git clone https://github.com/<username>/dotfiles.git ~/.dotfiles
```

---

Feel free to explore and adapt this configuration to suit your own workflow.
